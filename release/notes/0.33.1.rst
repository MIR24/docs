Эпик. MIRSCR-1261_
------------------------------------------
:term:`Админка`. Редактор просит добавить функционал текстовой трансляции

Реализована возможность редактировать любую новость в режиме трансляции.
При переходе в режим трансляции активируется поле, над основным тестом, в котором необходимо набить текущее состояние дел.
Следующее поле - временная метка, которая будет присвоена тексту выше. Ее можно изменить.
При нажатии СОХРАНИТЬ И ОСТАТЬСЯ метка + текст из первого поля добавляется перед основным текстом новости, новость сохраняется, редактор остается в режиме редактирования.
