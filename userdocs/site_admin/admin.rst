***********************************
Административный интерфейс «МИР24»
***********************************

| Данное руководство пользователя содержит инструкции по работе в административном интерфейсе (далее - Админка) новостного сайта «МИР24» (Далее - Сайт), инструкции разбиты по категориям.
| Категории разделяют основные направления администрирования:

 * Контент;
 * Управление;
 * Теги;
 * Служебные.

При переходе в Админку, по умолчанию открывается стартовая страница "Все публикации" (`editors.mir24.tv/all`_), где расположена лента всех опубликованных материалов. Для навигации по категориям и разделам Админки, необходимо открыть выдвигающееся Меню, доступное по кнопке "Меню" ( |menubtn| ) расположенное в шапке Админки:

.. image:: /images/admin/Admin-Head.png

Переход на стартовую станицу осуществляется по кнопке раздела: :guilabel:`&Меню` -->> :guilabel:`&Главная`
